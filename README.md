
# Dawsanm Realtors 🏡

A sleek, minimalist, and lightweight digital marketplace portal designed to bridge real estate and automotive commerce listings. Built entirely with a clean front-end architecture, this platform offers a fast, friction-free user experience for viewing and managing premium property and vehicle entries.

---

## ⚡ Features

*   **Dual Marketplace Listings:** Seamless integration for browsing both real estate properties and high-end automotive listings.
*   **Dynamic Client-Side Uploads:** Admins can instantly upload listings, handling image assets smoothly via Base64 data URLs directly in the browser.
*   **Live Filtering:** Clean, real-time search and filtering system for an organized user browsing experience.
*   **Local Persistence:** Built purely with a `localStorage` engine, allowing full data retention, dynamic updates, and admin verification without requiring a heavy backend database setup.
*   **Minimalist UI:** High-contrast design language prioritizing scannability, clean layouts, and modern presentation constraints.

---

## 🛠️ Tech Stack

*   **Frontend:** Semantic HTML5, Custom CSS3 Layouts
*   **Scripting Engine:** Vanilla JavaScript (ES6+)
*   **Data Management:** Browser `localStorage` API & Base64 Data Encoding
